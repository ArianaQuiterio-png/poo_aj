## Repositorio con ejercicios de Programación Orientada a Objetos en Python

## 1.  Crear el archivo .gitignore

Configurar el archivo .gitignore para restringir los archivos 
a sincronizar

````shell
*.pyc
__pyache__/
````

## 2. Ingresar archivos y carpetas

Permite ingresar todos los archivos y carpetas del proyecto, para identificar los cambios

```shell
git add .
````

## 3. Crear un punto de control con los cambios realizados en el proyecto

````shell
git commit -m "CREATED .gitignore"
````

* CREATED - Crear nuevos archivos o directorios
* UPDATED - Actualizar modulos del sistema
* fIXED - Corregir errores del sistema

## 4. Sincronizar los cambios con el repositorio

Sincronizar los cambios con el repositorio

````sell
git push -u origin main
````
